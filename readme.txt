These are the files for creating a responsive version of the Qompact "Touchscreen".
Project was begun on August 9, 2017. 
First commit completed on August 10, 2017.

Notes:
 - The circle graphics from the original version have been repaced by divs with a high border radius to make them appear circular.
 - Most of the re-sizing is done through "resize.js". 
 - Only the front page has been created at the time of first commit.
